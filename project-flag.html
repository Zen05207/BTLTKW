Flag Raising Simulation

Mô phỏng nghi lễ thượng cờ chân thực dành cho Roblox — kết hợp animation từ Blender (smartbone), xử lý trạng thái kéo cờ, chuyển sang cờ bay, và đồng bộ âm thanh.

Miêu tả chi tiết

Hệ thống mô phỏng quy trình kéo cờ gồm: lá cờ treo ban đầu, hệ thống ròng rọc kéo dần, khi tới vị trí xác định sẽ đổi thành lá cờ bay (với animation smartbone), đồng thời phát âm nhạc nghi lễ và hiệu ứng gió. Hỗ trợ xem nhiều người (multiplayer) với sync server-client cơ bản.

Tính năng chính

  • Quá trình kéo cờ mượt mà, có Tween/easing.
  • Chuyển đổi trạng thái cờ (hanging → flying) khi đạt chiều cao.
  • Hiệu ứng gió dùng xương (bone) để làm lá cờ bay tự nhiên.
  • Đồng bộ âm thanh nghi lễ (play/stop) khi kéo cờ.
  • Thiết kế UI hiển thị trạng thái (progress, time).

Công nghệ & công cụ

  • Roblox Studio (Lua: TweenService, RemoteEvent)
  • Blender (SmartBone / Rig) để tạo animation cờ
  • Audio: SoundService (Roblox) hoặc file .mp3 cho demo web
  • HTML/CSS/JS — trang demo & video minh họa

Tài liệu kỹ thuật / Ghi chú

  1. Để test multiplayer, triển khai script server kiểm tra và broadcast trạng thái (RemoteEvents).
  2. Tối ưu hóa mesh cờ và texture trong Blender để giảm tải khi import vào Roblox.
  3. Trong demo web: video lưu sẵn `video/flag.mp4`, poster ảnh `img/project1.jpg`.
